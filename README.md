### 概述

    wind-project是以学习和实际应用为目的而构建的一款由Felix独立开发的简约风的可定制化UI的多功能工具库。

	学习内容：独立开发，Rust+应用程序开发+不熟悉的技术+底层原理

- 前端：UI拖拽库的使用，前端性能优化，React进阶
- 应用：Rust语言入门
- 数据库：PGSQL，SQLite，Redis
- 项目：Git版本控制
